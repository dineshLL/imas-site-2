Help

If you have more questions, please refer to the following FAQ's

How to add a new connection

In the connections menu, Click on add new connection icon

Then, give connection parameters to MySQL and MongoDB servers accordingly

Test the connections and create the connection

How to Migrate a MySQL database to a Mongo database

Connect to the MySQL and Mongo servers via created connection

Right click on the MySQL database and select migrate

In the next interface, user can remove unnessasary tables or fields if needed

Then click next to continue

Next interface user can see how the mapping was done and if user needs to change the existing mapping decision, he can change it by toggle buttons

Then click continue to do modifications and give a name to migrated MongoDB schema

How to Create a new mongo collection

Select a database by right cliking on a particular Mongo database

Then, it will show exsisting collections

Then click on Create new collection and give a name to new collection
